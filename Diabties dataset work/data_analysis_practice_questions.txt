                          BONUS: KEY CONCEPTS GLOSSARY
                  (Quick Reference — Know These Cold)
================================================================================

STATISTICS
  - Mean, Median, Mode, Variance, Standard Deviation
  - Percentile, IQR (Interquartile Range)
  - Normal Distribution, Skewness, Kurtosis
  - Correlation (Pearson, Spearman), Causation
  - Hypothesis Testing: Null Hypothesis, p-value, α level
  - T-test, Chi-squared test, ANOVA
  - Confidence Interval, Statistical Power, Effect Size

DATA WRANGLING
  - Missing Value Types: MCAR, MAR, MNAR
  - Imputation: Mean, Median, Mode, KNN, Multiple Imputation
  - Outlier Detection: IQR Method, Z-score, Mahalanobis Distance
  - Data Normalization vs Standardization (MinMax vs Z-score)
  - Feature Engineering, Binning, One-Hot Encoding

SQL (Must Know for Interviews)
  - SELECT, WHERE, GROUP BY, HAVING, ORDER BY
  - JOINs: INNER, LEFT, RIGHT, FULL OUTER
  - Window Functions: ROW_NUMBER, RANK, LAG, LEAD, AVG OVER
  - Subqueries vs CTEs (WITH clause)
  - CASE WHEN for conditional logic

PYTHON / PANDAS
  - pd.read_csv(), df.head(), df.describe(), df.info()
  - df.groupby().agg(), df.pivot_table()
  - df.fillna(), df.dropna(), df.replace()
  - df.merge(), df.concat()
  - Visualization: matplotlib, seaborn (histplot, boxplot, heatmap, scatterplot)

BUSINESS / COMMUNICATION
  - Translating data to insights (not just numbers)
  - KPI definition and metric design
  - Dashboard design principles
  - Stakeholder storytelling
  - Data ethics and bias awareness
